## Local Setup and Usage Guide for Churn Prediction App

### Setup:

1. **Change into the cloned repository:**
    ```sh
    cd Churn-App-P4 
    ```

2. **Create a virtual environment:**
    ```sh
    python -m venv env
    ```

3. **Activate the virtual environment:**

    - **For Windows:**
        ```sh
        env\Scripts\activate 
        ```
    - **For Linux/macOS:**
        ```sh
        source env/bin/activate 
        ```
